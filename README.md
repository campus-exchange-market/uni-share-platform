# Uni Share Platform

## Proje Adı
Üniversite Öğrencileri İçin Akademik Gereç ve Ders Materyali Paylaşım Platformu

## Proje Açıklaması
Uni Share Platform, üniversite öğrencilerinin ders materyali, akademik kaynak ve ikinci el akademik gereçleri güvenli bir şekilde paylaşabilmesi için geliştirilen bir platformdur. Sistem; kullanıcı yönetimi, materyal paylaşımı, ilan oluşturma, mesajlaşma, bildirim ve ödül sistemi gibi modüllerden oluşmaktadır.

## Proje Yönetim Metodolojisi
Bu projede yazılım geliştirme metodolojisi olarak Scrum seçilmiştir. İş paketleri Jira Scrum Board üzerinde backlog, sprint ve görev durumları ile takip edilmektedir.

## Proje Yönetim Aracı
- Jira
- GitHub
- GitHub Pull Request ve Review süreçleri

## GitHub Süreci
Her ekip üyesi kendisine atanan Jira görevi için ayrı bir feature branch oluşturur. Değişiklikler commit edildikten sonra pull request açılır. Pull request, ekipten başka bir kişi tarafından review edilir ve onaylandıktan sonra main veya develop branch’ine merge edilir.
